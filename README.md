# ❄️ Welcome to Snowground

> A physics-driven sandbox where wind shapes every throw — knock out all the snowmen to win.

---

## 🏔️ About the Project

**Snowground** is an interactive sandbox environment built in Godot that simulates dynamic wind physics. Wander a snowy landscape, pick up and hurl snowballs influenced by a hidden wind force, and take down every snowman on the map to claim victory.

The environment is alive — snow falls from above, obscuring your vision and lending the world a genuinely wintry atmosphere. Footprints trail behind you as you move, then slowly fade away. Christmas trees dot the terrain and act as natural obstacles, breaking any snowball that hits them.

### Key Features

- **Wind Simulation** — A hidden wind value subtly curves every snowball you throw. Learn to read the environment and compensate.
- **Snowball Combat** — Hurl snowballs at snowmen to make them disappear. Hit all of them to win.
- **Fading Footprints** — Your path through the snow is visible but impermanent; footprints dissolve over time.
- **Falling Snow** — Atmospheric particle snow that adds visual depth and gently limits visibility.
- **Christmas Tree Obstacles** — Trees deflect snowballs on impact, adding a layer of spatial strategy.
- **Sandbox Feel** — No timers, no rigid objectives beyond finding and eliminating every snowman.

## 🛠️ How to Run

### Prerequisites

- **Godot Engine** — Download the latest stable version from [godotengine.org](https://godotengine.org/download)

### Steps

1. **Install Godot** using the link above (the latest stable release is recommended).
2. **Open the project** — Launch Godot, click *Import*, and navigate to the unzipped source folder. Select the `project.godot` file.
3. **Let assets import** — Godot will automatically process and import all project assets on first load. Wait for this to complete.
4. **Press Play** — Click the ▶ button in the top-right corner of the editor. The game will launch immediately.

> No additional dependencies or build steps are required.
